# Коллекция пользовательских сниппетов
Все скрипты запускаются через DevTools / Console (Desktop).

- [smartScroller](#smartscroller) — универсальный автоскроллер
- [vkPlaylistExporter](#vkplaylistexporter) — экспорт треков из ВК музыки в TXT документ
- [vkMassPlaylistAdder](#vkmassplaylistadder) — массовое добавление/перемещение треков в плейлисты 

## smartScroller
Универсальный инструмент автоматической прокрутки страниц вверх/вниз с поддержкой динамической подгрузки.  
Работает на любых сайтах, включая VK, Telegram Web, YouTube, маркетплейсы и бесконечные ленты.

Содержит встроенную панель управления и механизм выбора любого скролл-контейнера.

## vkPlaylistExporter
Экспортирует названия всех треков из любых разделов Мызыки ВК
Плейлисты, Мои треки, Поиск

Вывод сохраняется в TXT в формате: `Исполнитель - Название`.

## vkMassPlaylistAdder
Скрипт массового добавления музыки из Мои треки в плейлисты, перемещения треков между плейлистами.
Работает через окно «Редактировать плейлист» в VK и «Добавить аудиозапись».

Скрипт делает глубокую прокрутку, догружает весь список и автоматически проставляет все чекбоксы **снизу вверх**.
Не пропускает элементы, не трогает уже отмеченные.

Простое расширение Хром - https://github.com/TolyanDimov/vk-music-tools-ext
